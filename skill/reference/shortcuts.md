# Windows to Mac keyboard shortcuts

Source: switchtomac.app

The core idea: on a Mac you press Command (⌘) where you used to press Ctrl, and most shortcuts come straight back.

## The three modifier-key swaps
- Ctrl → ⌘ (Command). Command does what Ctrl did. Copy, paste, save, find: just slide your thumb one key over.
- Alt → ⌥ (Option). Option is your Alt: same place, same job, and it's also how you type accented characters.
- ⊞ Win → ⌘ (Command). The Windows key's jobs (search, run, lock) move to Command and to Spotlight (⌘ Space).

## Shortcuts by category

### Essentials
- Copy: `Ctrl C` → `⌘ C`.
- Cut: `Ctrl X` → `⌘ X`.
- Paste: `Ctrl V` → `⌘ V`.
- Paste without formatting: `Ctrl Shift V` → `⌥ ⇧ ⌘ V`.
- Undo: `Ctrl Z` → `⌘ Z`.
- Redo: `Ctrl Y` → `⇧ ⌘ Z`. Redo is Shift-Command-Z on a Mac.
- Select all: `Ctrl A` → `⌘ A`.
- Save: `Ctrl S` → `⌘ S`.
- Print: `Ctrl P` → `⌘ P`.
- Find on page: `Ctrl F` → `⌘ F`.
- Find next: `F3` → `⌘ G`.
- New: `Ctrl N` → `⌘ N`.
- Open: `Ctrl O` → `⌘ O`.
- Close the window: `Ctrl W` → `⌘ W`.
- Quit the whole app: `Alt F4` → `⌘ Q`. ⌘W closes one window; ⌘Q quits the app entirely.
- Bold: `Ctrl B` → `⌘ B`.
- Italic: `Ctrl I` → `⌘ I`.
- Underline: `Ctrl U` → `⌘ U`.

### Text & cursor
- Jump to start of line: `Home` → `⌘ ←`.
- Jump to end of line: `End` → `⌘ →`.
- Jump to start of document: `Ctrl Home` → `⌘ ↑`.
- Jump to end of document: `Ctrl End` → `⌘ ↓`.
- Move one word left: `Ctrl ←` → `⌥ ←`.
- Move one word right: `Ctrl →` → `⌥ →`.
- Select to start of line: `Shift Home` → `⇧ ⌘ ←`.
- Select to end of line: `Shift End` → `⇧ ⌘ →`.
- Select word by word: `Ctrl Shift ← / →` → `⇧ ⌥ ← / →`.
- Page up / down: `Page Up / Page Down` → `fn ↑ / ↓`.
- Delete forward: `Delete` → `fn ⌫`. On a Mac, ⌫ deletes backward (like Backspace). fn+⌫ deletes ahead. Control+D works too.
- Delete the previous word: `Ctrl Backspace` → `⌥ ⌫`.

### Windows & apps
- Switch between apps: `Alt Tab` → `⌘ Tab`.
- Switch windows of one app: `Alt + Tab cycles all` → `⌘ ``. The ` key sits just above Tab.
- Minimize the window: `⊞ Win ↓` → `⌘ M`.
- Hide the app: no Windows equivalent → `⌘ H`.
- Hide every other app: no Windows equivalent → `⌘ ⌥ H`.
- Force quit a frozen app: `Ctrl Alt Delete` → `⌘ ⌥ ⎋`. Opens the Force Quit window. (Ctrl-Alt-Del also covers lock/sign-out on Windows.)
- See all open windows: `⊞ Win Tab` → `⌃ ↑`. Mission Control, or swipe up with three or four fingers.
- See all windows of this app: no Windows equivalent → `⌃ ↓`.
- Move between desktops: `Ctrl ⊞ Win ← / →` → `⌃ ← / →`.
- Show the desktop: `⊞ Win D` → `fn F11`. Or press ⌘ with the Mission Control key.
- Full-screen the window: `F11` → `⌃ ⌘ F`. Or click the green button at the top-left of the window.

### Screenshots
- Capture the whole screen: `PrtScn` → `⌘ ⇧ 3`.
- Capture part of the screen: `⊞ Win Shift S` → `⌘ ⇧ 4`. Crosshair appears: drag to select, release to save to the Desktop.
- Capture a single window: `Alt PrtScn` → `⌘ ⇧ 4 then Space`. Press Space, then click the window you want.
- Screenshot & recording bar: `⊞ Win G` → `⌘ ⇧ 5`. All options in one place, including screen recording.
- Copy a shot to the clipboard: `Win + Shift + S copies` → `⌃ ⌘ ⇧ 4`. Add Control to any capture to copy it instead of saving a file.

### Files & Finder
- Open the file manager: `⊞ Win E` → `Finder, the smiley face in the Dock`. Finder is always running. ⌥⌘Space opens a Finder search window.
- New folder: `Ctrl Shift N` → `⌘ ⇧ N`.
- Rename a file: `F2` → `⏎`. Select the file, then press Return.
- File details (Properties): `Alt ⏎` → `⌘ I`. Called “Get Info” on a Mac.
- Move a file to the bin: `Delete` → `⌘ ⌫`.
- Empty the bin: no Windows equivalent → `⌘ ⇧ ⌫`.
- Preview a file instantly: no Windows equivalent → `Space`. Quick Look: select a file and tap Space. A Mac favorite.
- Move a file (cut & paste): `Ctrl X / Ctrl V` → `⌘ C then ⌥ ⌘ V`. Copy first, then Option-Command-V moves it instead of copying.

### System & search
- Search & launch anything: `⊞ Win` → `⌘ Space`. Spotlight replaces the Start menu and Run box: type to find apps, files, math, even clipboard history.
- Open Settings: `⊞ Win I` → `Apple menu ▸ System Settings`.
- Lock the screen: `⊞ Win L` → `⌃ ⌘ Q`.
- Log out: no Windows equivalent → `⇧ ⌘ Q`.
- Emoji picker: `⊞ Win .` → `fn E`. Or Control-Command-Space.
- Accents & special characters: no Windows equivalent → `hold ⌥, or press fn twice`. Hold a letter for a popup of accents, or use ⌥ combos (⌥e then e → é).
- The full list of apps: `Start menu, then scroll` → `fn ⇧ A`. Opens “Applications.” (Launchpad was removed in macOS Tahoe.)

### Browser
- New tab: `Ctrl T` → `⌘ T`.
- Reopen the last closed tab: `Ctrl Shift T` → `⌘ ⇧ T`.
- Close the tab: `Ctrl W` → `⌘ W`.
- Jump to the address bar: `Ctrl L` → `⌘ L`.
- Reload the page: `F5` → `⌘ R`.
- Hard reload: `Ctrl F5` → `⌘ ⇧ R`.
- Zoom in / out: `Ctrl + / −` → `⌘ + / −`.
- Next / previous tab: `Ctrl Tab` → `⌃ Tab`. Or ⌘1–9 to jump straight to a tab.
- Bookmark this page: `Ctrl D` → `⌘ D`.
- Downloads: `Ctrl J` → `⌘ ⌥ L`. That's Safari; in Chrome it's ⇧⌘J.
