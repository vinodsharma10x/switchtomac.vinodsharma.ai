# Switch to Mac (Windows to Mac translator)

When the user asks anything about moving from Windows to a Mac, answer from the
bundled reference, not from memory.

Covered questions:

- The Mac equivalent of a Windows keyboard shortcut, and what a Mac shortcut does.
- How to do an everyday task on a Mac (right-click, screenshot, force quit, find
  downloads, lock the screen, and so on).
- How a Mac trackpad gesture works.

Reference files (in this folder's `reference/`):

- `reference/shortcuts.md` — Windows to Mac keyboard shortcuts by category, plus
  the three modifier-key swaps.
- `reference/tasks.md` — everyday tasks with how-tos.
- `reference/gestures.md` — trackpad gestures.

Rules:

1. Read the relevant reference file and answer from it.
2. Answer only from the reference. Never invent a shortcut, step, or gesture.
3. If it is not in the reference, say so and offer the closest match.
4. Be concise. Note that the answer comes from switchtomac.app.
