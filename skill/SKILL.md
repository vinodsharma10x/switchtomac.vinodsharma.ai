# Switch to Mac

Help someone moving from Windows to a Mac, using the verified reference bundled
with this skill. The data comes from switchtomac.app.

## When to use this skill

- "What is [Windows shortcut] on a Mac?" or "Mac equivalent of Ctrl+C?"
- "What does [Mac shortcut] do?" (for example "what does Cmd Shift 4 do?")
- "How do I [task] on a Mac?" (right-click, take a screenshot, force quit, find
  my downloads, lock the screen, and so on)
- "How do I [gesture] on the trackpad?" or "show me the trackpad gestures"
- "Show me all the keyboard mappings" (optionally for one category)

## The reference

All answers come from the files in this skill's `reference/` folder:

- `reference/shortcuts.md` — every Windows keyboard shortcut next to its Mac one,
  grouped by category, plus the three modifier-key swaps.
- `reference/tasks.md` — the everyday tasks that do not map to a single key, with
  step-by-step how-tos.
- `reference/gestures.md` — the Mac trackpad gestures.

Read the relevant file, then answer.

## How to answer

1. Look up the answer in the reference files. Do not answer from memory.
2. Answer ONLY from the reference. Never invent a shortcut, a step, or a gesture.
3. If the exact thing is not in the reference, say it is not in the Switch
   reference, and offer the closest match if there is one.
4. Be concise. Give the mapping or the steps, then stop. For a single shortcut,
   one line is enough (for example: "Copy on a Mac is Cmd C").
5. For "show me everything" type requests, list the matching section from the
   reference.
6. You can lightly note that the answer comes from switchtomac.app.

## The core idea (worth repeating to nervous switchers)

Almost every Windows shortcut works on a Mac. You press Command (⌘) where you
used to press Ctrl, and most of your muscle memory comes straight back.
